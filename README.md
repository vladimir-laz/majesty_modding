# Majesty 2 — моддинг

Коллекция модов для **Majesty 2: The Fantasy Kingdom Sim**, их документация
и объединённая сборка.

## Структура репозитория

- **`majesty_mods/`** — исходные моды (каждый в своей папке с `resource.mod`):
  - `10х_скорость` — множители скорости игры до 10×
  - `Второй стражник` — второй лучник в сторожевых башнях со 2-го уровня
  - `элитные башни` — 3 уникальные башни в меню замка
  - `Рыцарь` — герой `hero_knight` в гильдии воинов
  - `4 кузница у монстров` — 4-й уровень оружия/брони героям-монстрам
  - `мини-сборка` — логова монстров, гильдии, спавн

- **`all_mods/`** — объединённая сборка модов в один `resource.mod`
  (+ инструкция и описание мерджа). Готова к установке. Включает все моды,
  **кроме «4 кузница у монстров»** (исключена из-за проблем при запуске).
- **`all_mods.zip`** — архив сборки (в корне — `resource.mod`).
- **`obsidian-vault/`** — граф знаний по моддингу Majesty 2: форматы файлов,
  игровые механики, разбор каждого мода и практические рецепты
  («что в каком файле менять»). Открывается в [Obsidian](https://obsidian.md).

## Установка любого мода

Переместить папку `resource.mod` из мода (или из `all_mods/`) в каталог с
игрой. Удаление — удалить `resource.mod`. Подробнее — в
`obsidian-vault/Общее/установка-и-совместимость.md`.

## С чего начать в документации

Открой `obsidian-vault/` в Obsidian и начни с `README.md` графа знаний.
Для конкретных правок — раздел **Рецепты** с мастер-картой файлов.
