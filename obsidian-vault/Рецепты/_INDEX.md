## Мастер-карта: что где менять
| Хочу изменить… | Файл | Блок / поле |
|---|---|---|
| Стоимость **постройки здания** | [[shops-set\|shops.set]] | `castle → buildings → "<здание>" → Cost → "count money"` |
| Лимит зданий / потолок апгрейда | [[shops-set\|shops.set]] | там же: `"Max count"`, `"Max upgrade"` |
| Условие апгрейда здания | [[shops-set\|shops.set]] | там же: `"Upgrades prerequsity"` |
| Стоимость **найма героя** | [[shops-set\|shops.set]] | `"<гильдия>" → units → "<hero>" → Cost → "count money"` |
| **Добавить героя** в здание | [[shops-set\|shops.set]] + меню + [[loctable-xml\|loctable]] | `units` гильдии + кнопка в `*_menu.xml` |
| Урон оружия / защита брони | [[unit-decour-xml\|unit_decour.xml]] | `Weapon → Params → f_dps_*` / `f_defence_*` |
| **Характеристики героя (сила/ловкость/ум/выносливость)** | [[rpg-params-xml\|rpg_params.xml]] | секция `<Heroes>`: `f_strength/f_agility/f_intellect/f_stamina` (0–20) |
| Базовые HP/урон героя по уровням | [[rpg-params-xml\|rpg_params.xml]] | `f_maxHealth`, `f_dps_*` в `<Heroes>` |
| Стоимость/кулдаун/мощь заклинаний | `gameData/magic/spells.xml` † | по [[карта-игровых-файлов]] |
| Бонусы характеристик от предмета | [[unit-decour-xml\|unit_decour.xml]] | `Weapon → Params → f_strength` и др. |
| Доступный уровень снаряжения | [[inventions-graphml\|inventions.graphml]] + [[shops-set\|shops.set]] | узел изобретения + `weapon`/`armour` |
| Скорость/обзор/способности героя | `<hero>.def` | `extenders/mover.inc`, `sensor.inc`, `actions.inc` |
| Кого спавнит здание/башня | [[def-сущности\|.def]] + [[global-spawn-settings\|spawn.set]] | `spawner_extender` + программа спавна |
| Видимые тексты (имена/описания) | [[loctable-xml\|loctable]] | ключи `#KEY` |
| Иконки кнопок | [[texture-units-xml\|texture_units.xml]] + `.dds` | `<Texture>` |
| Скорость игры | [[game-window-xml\|game_window.xml]] | `COUNTER_game_speed → s_counter_values` |

Полная карта расположения ВСЕХ игровых конфигов (не только из наших модов) —

## Рецепты
- [[изменить-стоимость-здания]] — цена постройки, лимиты, условия апгрейда
- [[изменить-стоимость-героя]] — цена найма, лимит, время найма
- [[добавить-героя-в-здание]] — добавить нанимаемого героя в гильдию (как мод «Рыцарь»)
- [[изменить-статы-героя]] — что из статов реально моддится и где (+ оговорка про RPG-класс)
- [[изменить-урон-и-броню]] — DPS оружия и защиту брони в `unit_decour.xml`
- [[максимальные-характеристики-героев]] — всем героям сила/ловкость/ум/выносливость = 20 (`rpg_params.xml`)
- [[награда-за-убийство-монстров]] — золото героям за убийство монстров (`TreasureKeeper` в дефах монстров; напр. ×4)
- [[увеличить-спавн-логов]] — больше монстров из логов, построенных игроком (global_spawn_settings.set)
- [[спавн-и-деньги-в-миссиях]] — ⭐ спавн логов кампании и стартовые деньги задаются в файле МИССИИ (.mi): `helper/spawn_program_collection` (task none/protect = гарнизон, attack_in_zone = волны, флаг First wave = старт) и `{"kingdoms"}` (деньги игрока = humanplayer 1). Миссия также решает, какие здания доступны.
- [[добавить-монстрское-здание-людям]] — гильдия оборотней / кузница монстров (`mks_*`) в человеческом замке (shops.set + castle_menu; сущность/текстура из базы)
- [[крестьяне]] — число крестьян в доме (`peasant_spawner` max count) и почему HP крестьян пока не поменять

## Общий принцип
Любая правка = положить **полную копию** нужного файла в зеркальный путь
внутри `resource.mod` и отредактировать (см. [[система-модов]]). После —
свериться с [[установка-и-совместимость]] на конфликты.
