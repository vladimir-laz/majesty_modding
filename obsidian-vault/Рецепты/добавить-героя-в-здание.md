# Рецепт: добавить героя в здание (гильдию)

Эталон — мод [[рыцарь]]: он добавил `hero_knight` в `guild_warrior`. Ниже
обобщённый порядок. Различают **два случая**.

## Случай A: существующий герой в новое здание
Если герой уже есть в игре (`<hero>.def` существует), достаточно
**данных + кнопки**:

1. **[[shops-set|shops.set]]** — в блок нужной гильдии, в `units`, добавить:
   ```
   {"hero_knight"
       {Cost {money} {"count money" 650}}
       {Entity hero_knight}
       {"Max count" 2}
       {"Production time" 10}}
   ```
2. **Меню гильдии** `interface/dynamic/menu/<гильдия>_menu.xml`
   ([[интерфейс-меню-xml]]) — добавить кнопку найма:
   ```xml
   <Button_2 action="RecruitHeroAction" stateAction="CheckRecruitAction">
     <appearance texture="BUT_hire_Knight" autoTexture="TEX_MASK"
                 propertyBlock="action_but" userdata="hero_knight" />
   </Button_2>
   ```
   `userdata` = id героя (совпадает с `Entity`); `texture` — иконка кнопки.
3. **Текстура кнопки** (если новая) — объявить в [[texture-units-xml]] и
   положить `.dds`.

## Случай B: совсем новый герой
Дополнительно к шагам A нужно **создать саму сущность героя**:

4. **`<hero>.def`** в `entity/units/heroes/<hero>/` ([[def-сущности]]) —
   обычно копируют похожего героя и правят: классы (`entity_classes.inc`),
   стартовое оружие/броню (`inventory.inc`), способности (`actions.inc`),
   `RPGModule` (RPG-класс), скорость/обзор. См. [[статы-героев]].
5. **Способности/умения** героя — в той же гильдии в [[shops-set|shops.set]]
   в `inventions` (как `knight_imp_attack`, `knight_swordmaster_buff`…),
   а в меню — кнопки `action="InventAction"` с `userdata="<skill>"`.
6. **Снаряжение** — оружие/броня в [[unit-decour-xml]] + изобретения в
   [[inventions-graphml]] + записи `{weapon}`/`{armour}` в `shops.set`.
7. **Локализация** — [[loctable-xml]]: `#HERO_<X>_NAME`, `_DESC`, имена
   умений/состояний.

## Проверочный список связей
кнопка(`userdata`) → запись в `units`(`Entity`) → `<hero>.def` →
RPG-класс/оружие; иконка → texture_units → `.dds`; имя → loctable `#KEY`.
Если что-то из цепочки отсутствует — кнопка серая/без иконки/без имени.
