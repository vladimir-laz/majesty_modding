# Рецепт: изменить урон оружия и защиту брони

## Файл
`gameData/units/unit_decour.xml` ([[unit-decour-xml]]).

## Где править
Нужный предмет в `<WeaponGroup>` по `<s_name>`, секция `<Params>`:
```xml
<Weapon>
  <s_name>hero_archer_bow_02</s_name>
  <f_cost>30</f_cost>
  <Params>
    <f_dps_range>8.1</f_dps_range>   <!-- ← урон, менять здесь -->
  </Params>
</Weapon>
```

## Что менять
- **Урон**: `f_dps_hth` (ближний), `f_dps_range` (дальний),
  `f_dps_magic` (магия).
- **Защита** (у брони): `f_defence_hth`, `f_defence_range`,
  `f_defence_magic`.
- **Бонусы**: `f_strength`, `f_agility`, `f_intellect`, `f_stamina`,
  `f_maxHealth`, `f_maxMana`, `f_movement`.
- **Частота атак**: `f_frequencyFactor` (база 1).
- **Цена предмета**: `<f_cost>`.

Поля, которых нет в `<Params>`, считаются 0 (для `f_frequencyFactor` — 1) —
чтобы добавить бонус, просто допиши нужный тег внутрь `<Params>`.

## Шаги
1. Скопировать `unit_decour.xml` в `resource.mod/gameData/units/`.
2. Найти предмет по `<s_name>` (имена вида `<hero>_<тип>_<NN>`, где `NN` —
   уровень; `weapon05`/`bow05` = уровень 4).
3. Поменять/добавить значения в `<Params>`. Кодировка файла —
   windows-1251 (сохранять в ней же).

## Связь
Какой предмет носит герой — задаётся в его `.def` (`m2inventory`); какой
уровень доступен в кузнице — через [[inventions-graphml]] + [[shops-set]].
Поднять урон только у топового уровня = править `*_05`/`*_4`.
