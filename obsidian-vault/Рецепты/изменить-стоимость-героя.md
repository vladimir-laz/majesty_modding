# Рецепт: изменить стоимость найма героя

## Файл
`set/trading/shops.set` ([[shops-set]]).

## Где править
Блок **гильдии** (не замка!) → список `units` → нужный герой:
```
{"guild_warrior"
    {entity_id guild_warrior}
    {units
        {"hero_warrior"
            {Cost {money} {"count money" 200}}   ; ← ЦЕНА найма
            {Entity hero_warrior}
            {"Max count" 3}                        ; ← лимит этого героя
            {"Production time" 10}                 ; ← время найма
        }
        {"hero_knight"
            {Cost {money} {"count money" 650}}
            {Entity hero_knight}
            {"Max count" 2}
            {"Production time" 10}
        }
    }
    {inventions { ...умения гильдии... }}
}
```

## Что менять
- **Цена найма** — `{"count money" N}`.
- **Лимит** этого героя — `{"Max count" N}`.
- **Время найма** — `{"Production time" N}`.

## Шаги
1. Скопировать `shops.set` в `resource.mod/set/trading/`.
2. Найти **блок гильдии** по `{entity_id <гильдия>}` (например
   `guild_warrior`, `guild_hunter`, `guild_mage`). Внимание: имя гильдии
   встречается в файле и в `castle.buildings` (цена постройки) — нужен блок
   с `{units ...}`.
3. Внутри `units` найти героя и поменять `count money`.

## Не путать
- Цена **постройки** самой гильдии → [[изменить-стоимость-здания]].
- Это цена **золотом за найм** конкретного героя.
