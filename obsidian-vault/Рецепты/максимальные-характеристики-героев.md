# Рецепт: максимальные характеристики у всех героев

Цель: у всех героев сила/ловкость/ум/выносливость = максимум (20).

## Файл
`gameData/units/rpg_params.xml` ([[rpg-params-xml]]). В базовой игре — внутри
`resource.pak` (в Collection155 распакован в `resource/gameData/units/`).
Кладём отредактированную копию в `resource.mod/gameData/units/rpg_params.xml`.

## Где править
Только секция **`<Heroes>`** (классы героев). В каждом `<Params>`:
```xml
<f_strength>20</f_strength>
<f_agility>20</f_agility>
<f_intellect>20</f_intellect>
<f_stamina>20</f_stamina>
```
Секции `<Citizens>/<Monsters>/<Buildings>` НЕ трогать (иначе усилишь и врагов).

## Шаги
1. Достать `rpg_params.xml` (из `resource.pak` 7-Zip'ом, либо из
   распакованной `resource/gameData/units/`).
2. В пределах `<Heroes>…</Heroes>` заменить значения 4 атрибутов на 20.
   ⚠️ Значения бывают целые (`14`) и дробные (`14.0`) — замена должна ловить
   оба (`[\d.]+`), иначе часть героев (`mks_hero_*`, `hero_high_*`) не
   обновится (всего классов 31).
3. Положить файл в `resource.mod/gameData/units/rpg_params.xml`.
4. Проверить: 31 герой × 4 атрибута = 124 значения «20»; теги `<Params>`
   сбалансированы; монстры (напр. `beast_rat`) не изменены.
