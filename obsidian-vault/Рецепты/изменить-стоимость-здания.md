# Рецепт: изменить стоимость / лимиты постройки здания

## Файл
`set/trading/shops.set` ([[shops-set]]).

## Где править
Блок замка → список `buildings` → нужное здание:
```
{"castle"
    {buildings
        {"guild_warrior"
            {Cost {money} {"count money" 550}}   ; ← ЦЕНА постройки
            {Entity guild_warrior}
            {"Max count" 3}                        ; ← сколько можно построить
            {"Max upgrade" 3}                      ; ← потолок уровня
            {"Upgrades prerequsity"                ; ← условия апгрейда
                {"2" {buildings "castle_3 guild_mage_3"}}}
        }
    }
}
```

## Что менять
- **Цена** — число в `{"count money" N}`.
- **Лимит штук** — `{"Max count" N}`. Для «без лимита» ставят `2147483647`
  (или большое число — так у [[элитные-башни]]).
- **Потолок уровня** — `{"Max upgrade" N}`.
- **Условие апгрейда** — `{"Upgrades prerequsity" {"<уровень>" {buildings
  "A_3 B_3"}}}` (суффикс `_2`/`_3` = уровень требуемого здания).

## Шаги
1. Скопировать оригинальный `shops.set` в
   `resource.mod/set/trading/shops.set` (если ещё не там).
2. Найти нужное здание в `castle.buildings` по имени.
3. Поменять значение, сохранив формат [[set-конфиги|вложенных `{}`]].
4. Проверить конфликты по [[установка-и-совместимость]] (файл правят 4 мода).

## Не путать
Цена **постройки гильдии** — здесь (в `castle.buildings`). Цена **найма
героя** внутри гильдии — в другом блоке (`"<гильдия>".units`), см.
[[изменить-стоимость-героя]].
