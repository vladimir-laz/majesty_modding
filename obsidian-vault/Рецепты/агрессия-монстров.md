# Рецепт: повысить агрессию монстров

## Где
В **дефе монстра** (`entity/units/monsters/<имя>/<имя>.def`, расплющенный —
инлайн из `extenders/ai.inc`), блок `{"relations"}` внутри
`Behaviour.HeroMonsterBehaviour → HeroMonsterTargetManager`:
```
{"Behaviour.HeroMonsterBehaviour"
    {HeroMonsterTargetManager
        {AllureFactors }            ; обычно пусто
        {"relations"
            "hero"   5              ; вес тяги атаковать героев
            "castle" 5
            "guild"  5
            "peasant" 5  "guard" 5  "market" 5  "smithy" 5  "peasant_house" 5 ...
        }
    }
}
```
**Чем больше положительный вес — тем агрессивнее** монстр лезет на эту цель
(героев, здания, жителей, экономику). Отрицательные (`"citizen" -20`) = избегать.

## Как
Домножить положительные веса (×2 = «немного, но заметно»). Отрицательные не
трогать. Игра грузит расплющенный `.def`, поэтому править `.def` (не `ai.inc`).
Идемпотентно: брать relations из БАЗЫ, ×2, подставлять в билд (в билде у монстров
ещё правка золота `TreasureKeeper` — её relations не задевает).
```python
re.sub(r'"(\w+)"\s+(-?\d+(?:\.\d+)?)', boost_positive, relations_block)
```

## Сделано в сборках
×2 положительным relations у 36 дефов монстров (из 94; 44 имеют блок, часть —
только нули/отрицательные). Напр. `beast_werewolf`: hero/castle/guild 5→10.
