# База знаний: моддинг Majesty 2

Граф знаний по модификациям игры **Majesty 2: The Fantasy Kingdom Sim**.
Описывает форматы игровых файлов, игровые механики, которые эти файлы
задают, и конкретные моды из папки `majesty_mods/`.

Последнее обновление: 2026-06-05

## Как пользоваться

Заметки атомарны (одна заметка = одна тема) и связаны вики-ссылками `[[...]]`.
В каждой папке есть `_INDEX.md` с аннотированным списком заметок. Открой
граф Obsidian, чтобы увидеть связи визуально.

**С чего начать:**
- **Нужно что-то конкретное поменять** (цена, герой, статы) → раздел
  [[Рецепты/_INDEX|Рецепты]] с мастер-картой «что в каком файле».
- **Где вообще лежит какой конфиг игры** → [[карта-игровых-файлов]]
  (полная карта, включая `spells.xml`, `rpg_params.xml` и др.).
- **Внешние гайды/форумы, инструменты, известные баги** → [[ссылки-и-источники]].
- Новичку в моддинге → [[система-модов]], затем [[процесс-моддинга]].
- Нужно понять конкретный файл → раздел [[Форматы/_INDEX|Форматы]].
- Нужно понять, как устроена игровая система → [[Механики/_INDEX|Механики]].
- Нужно понять, что делает конкретный мод → [[Моды/_INDEX|Моды]].

## Разделы

### [[Рецепты/_INDEX|Рецепты]] ⭐
Практические инструкции «как сделать X» + **мастер-карта «что в каком
файле менять»**. Главный рабочий раздел: изменить стоимость зданий/героев,
добавить героя в здание, поменять статы героя, урон оружия и броню.

### Готовые сборки
Что собрано и что в каждом билде — [[сборки]]. Билды локальные (в git не пушатся).

### [[Общее/_INDEX|Общее]]
Как вообще устроена система модов Majesty 2: папка `resource.mod`,
механизм переопределения файлов, установка, совместимость, рабочий
процесс модостроителя.

### [[Форматы/_INDEX|Форматы]]
Справочник по форматам файлов: `.def` (сущности), `.set` (конфиги:
`shops.set`, `global_spawn_settings.set`), интерфейсные `.xml` (меню,
`game_window.xml`), `.loctable.xml` (локализация), `.graphml` (дерево
изобретений), `.inc` (экстендеры), бинарные ассеты.

### [[Механики/_INDEX|Механики]]
Игровые системы, которые задаются файлами: спавн существ, меню постройки
и апгрейды, башни и оборона, изобретения/кузница, герои и гильдии, логова
монстров, скорость игры.

### [[Моды/_INDEX|Моды]]
Разбор каждого мода из коллекции: что меняет, какие файлы трогает,
конкретные значения.
