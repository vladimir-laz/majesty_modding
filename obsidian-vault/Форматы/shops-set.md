# `shops.set` — меню постройки в замке

## Описание
`set/trading/shops.set` — что и почём можно **строить/исследовать/нанимать**
в замке. Формат — [[set-конфиги|вложенные `{}`]]. Это самый «конфликтный»
файл коллекции: его правят 4 мода ([[рыцарь]], [[элитные-башни]],
[[4-кузница-у-монстров]], [[мини-сборка]]). Файлы большие (сотни КБ).

## Два разных вида блоков (ВАЖНО)
В файле несколько «магазинов» верхнего уровня, и **цена постройки здания**
и **цена найма героя** живут в РАЗНЫХ блоках:

1. **Замок** `{"castle" ... {buildings {...}}}` — что и почём **строить**.
   Здесь же `{inventions ...}` замка.
2. **Гильдия/кузница** `{"guild_warrior" {entity_id guild_warrior}
   {units {...}} {inventions {...}}}` — магазин конкретного здания:
   - `{units ...}` — каких **героев нанимать** (цена найма, лимит).
   - `{inventions ...}` — какие **умения/предметы исследовать**.

То есть `guild_warrior` встречается дважды: как строка в `castle.buildings`
(цена постройки гильдии) и как самостоятельный блок с `units`/`inventions`
(что гильдия предлагает). Не перепутать!

## Блок здания
```
{"guild_warrior"
    {version original}
    {Cost {money} {"count money" 550}}     ; цена постройки
    {Entity guild_warrior}                  ; какую сущность строит (.def)
    {"Max count" 3}                         ; лимит штук
    {"Max upgrade" 3}                       ; до какого уровня улучшается
    {Prerequesities {gold 0} {"heroes count" 0}}
    {"Upgrades prerequsity"
        {"2" {buildings "castle_3 guild_mage_3"}}  ; условие апгрейда до ур.2
    }
}
```

Ключевые поля:
- `{"count money" N}` — стоимость в золоте.
- `{"Max count" N}` — сколько можно построить. `2147483647` или
  `123234535` = практически без лимита (см. [[элитные-башни]]).
- `{"Max upgrade" N}` — потолок уровня здания.
- `{"Upgrades prerequsity" {"<уровень>" {buildings "A B"}}}` — какие здания
  (с уровнем через суффикс `_2`/`_3`) нужны для апгрейда.

## Блок наёма героя (внутри `{units ...}` гильдии)
```
{"hero_knight"
    {version original}
    {Cost {money} {"count money" 650}}   ; ← цена НАЙМА героя
    {Entity hero_knight}                  ; какую сущность нанимает (.def героя)
    {"Max count" 2}                       ; сколько таких героев держать
    {"Production time" 10}                ; время найма
}
```
Лежит в `{"<гильдия>" ... {units { ЗДЕСЬ }}}`. Чтобы **добавить героя в
здание**, добавляют такой блок в `units` нужной гильдии (см.
[[добавить-героя-в-здание]]).

## Блок изобретения / оружия / брони
```
{inventions { {"magic_dest" {invention magic_dest} {prerequsity {...}}} }}
...
{weapon { {"hero_knight_sword_04" ...} }}
{armour { {"hero_knight/4" ...} }}
```
Оружие/броня привязаны к изобретениям (`{invention mks_swords_4}`), у
многих `{Production time 0}` (мгновенно). См. [[изобретения-кузница]].
