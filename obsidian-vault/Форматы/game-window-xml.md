# `game_window.xml` — окно игры (и панель скорости)

## Описание
`interface/enGUIne/Skins/General/Windows/game_window.xml` — описание
главного игрового окна (HUD): мини-карта, панели, ресурсы и др. Очень
большой файл (~27 000 строк). В отличие от меню зданий
([[интерфейс-меню-xml]]), здесь скин-разметка элементов окна:
именованные элементы `<s_name>...</s_name>`, размеры, координаты, текстуры.

Для модов важна **панель управления скоростью игры**.

## Панель скорости
Иерархия элементов (имена `s_name`):
- `GAME_SPEED_CTRL` — контейнер управления скоростью (правый край HUD).
- `panel_game_speed` — вложенная панель.
- `fr_caption_game_speed` — фон-рамка (`M2_FR_ROUND_BG`).
- `COUNTER_game_speed` — **счётчик-переключатель** множителя скорости
  (кнопки −/+, `HUM_SLIDER_BUTL_N` / `HUM_SLIDER_BUTR_N`).

Множители задаются строкой значений счётчика:
```xml
<s_counter_values>0.2x,0.5x,1x,2x,5x,10x</s_counter_values>
```
- Список значений = доступные скорости. Именно сюда [[10x-скорость]]
  добавляет `5x` и `10x`.
- `b_cycled=false` — счётчик не зацикленный (на `10x` не перескакивает в
  начало).
- Подсказка кнопки — ключ локализации `#uiSpeedChange`.

## Механизм
XML только описывает **UI-виджет** выбора множителя. Само масштабирование
времени выполняет движок игры (C++): при выборе значения движок применяет
соответствующий коэффициент к игровому времени. Мод не содержит логики —
он лишь расширяет список доступных значений в виджете. См.
[[скорость-игры]].
