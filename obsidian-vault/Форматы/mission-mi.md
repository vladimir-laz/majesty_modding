# Формат файла миссии `.mi`

## Описание
`resource/map/<кампания>/<миссия>/<миссия>.mi` — файл миссии (карта + вся
её логика). Формат — [[set-конфиги|вложенные `{}`]]. Большой (~600 КБ–1 МБ,
до ~25 тыс. строк). **Миссия переопределяет очень многое**, чего нет в
глобальных файлах мода: спавн логов, стартовые деньги, доступность построек,
расстановку сущностей. Файлов ~90 (кампании + дополнения + мультиплеер),
суммарно ~54 МБ — поэтому в мод их кладут только при необходимости.

## Ключевые блоки

### Сущности на карте
```
{entity "logovo_wolf" 0x89
    {position -84 68}
    {"level" {lastlevelupdatedview 1}}
    {"entitynationality" {fatherland {kingdomdescriptor "enemy"} {humanplayer 0}}}
}
```
- `kingdomdescriptor` — `"player"` / `"enemy"` / `"neutral"`.
- `{flags "randomizable"}` / `"super_randomizable"` — рандомизация **позиции/
  контента** логов и сундуков (фича патча 1.3), НЕ количество монстров.

### Спавн монстров — `helper/spawn_program_collection`
**Главное:** спавн логов кампании задаётся ЗДЕСЬ (per-mission), а НЕ в
глобальном [[global-spawn-settings]]. Программа:
```
{"wolves"
    {raw beast_wolf}
    {"AI task" {task protect}{target_tag logo_wolves}}
    {count "1 1"} {"spawn time" 45} {"max count" 3}
    {"First wave"}
}
```
- **`AI task` → `task`**: `protect`/`none` = **стационарный гарнизон логова**
  (это «монстры в логове»); `attack_in_zone` = **волны атаки** на игрока
  (часто count/max 20–30); `waypoints` = патрули.
- `count "min max"` — сколько за один спавн (магнитуда залпа).
- `"spawn time"` — период; `"max count"` — потолок одновременно живых.
- **`{"First wave"}`** — программа вываливает свою пачку **сразу на старте**
  миссии. Если у неё большой `count` → «лавина в день 1». Чтобы усилить лога
  без раннего перебора: `count` оставить маленьким (мягкий старт), а
  `max count` поднять (лог дорастёт постепенно).

### Стартовые деньги — `{"kingdoms"}`
```
{kingdom {kingdomdescriptor {kingdomdescriptor "player"} {humanplayer 1}}
    {kingdomtreasure "money" 2000 "wood" 0 "id" 1}    ; деньги ИГРОКА
}
```
Деньги игрока = `kingdomtreasure "money" N` в королевстве с **`{humanplayer 1}`**
(ориентир — humanplayer, НЕ `id`: он разный в разных миссиях). Враги —
`humanplayer 0`, не трогать.

### `sp_global_executor`
Сохранённое состояние городских спавнеров (peasant_house, taxer, guard…) —
с пустым `spawned_entities`. Параметры берутся из глобальных программ.

## Что ещё контролирует миссия
**Доступность построек.** Здание есть в [[shops-set|shops.set]] мода, но в
миссии его не построить, если миссия/кампания не включила его в свой список
(поэтому, напр., элитные башни не видны в части кампанийных миссий).
