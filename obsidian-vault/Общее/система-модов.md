# Система модов Majesty 2 — `resource.mod`

## Суть
Мод для Majesty 2 — это папка **`resource.mod`**, которую кладут в каталог
с установленной игрой. Внутри она **зеркалит структуру ресурсов игры**:
движок при загрузке ресурса сначала ищет файл в `resource.mod`, и только
если не находит — берёт оригинал из базовых ресурсов игры. Таким образом
любой файл мода **переопределяет** (overrides) одноимённый файл игры.

Ключевое следствие: чтобы что-то изменить, мод **не патчит** оригинал, а
кладёт рядом **полную копию файла** с внесёнными правками. Поэтому даже
правка одного числа требует положить весь файл целиком (см. большие
`shops.set` на сотни КБ).

## Структура папки мода
Внутри `resource.mod` повторяется дерево игры:

```
resource.mod/
├── entity/               сущности игры
│   ├── buildings/<name>/<name>.def      здания (+ .mdl модель, extenders/*.inc)
│   ├── units/monsters/<name>/<name>.def монстры
│   ├── units/heroes/<name>/             герои
│   └── items/.../<weaponNN>/            предметы (оружие/броня)
├── gameData/
│   ├── spawn/global_spawn_settings.set  настройки спавна существ
│   ├── inventions/inventions.graphml    дерево изобретений
│   └── units/unit_decour.xml            параметры юнитов/декора
├── set/trading/shops.set                меню постройки в замке
├── interface/
│   ├── dynamic/menu/*.xml               меню зданий (кнопки действий)
│   └── enGUIne/...                       скины окон, текстуры, texture_units
└── localization/M2_mod.loctable.xml     строки локализации
```

## Из чего «собран» мод
Большинство модов комбинируют несколько слоёв изменений:
1. **Данные сущности** — [[def-сущности|.def]] здания/юнита (статы, экстендеры).
2. **Экономика/доступность** — [[shops-set|shops.set]] (стоимость, лимиты, требования).
3. **Поведение/спавн** — [[global-spawn-settings|global_spawn_settings.set]].
4. **Интерфейс** — [[интерфейс-меню-xml|меню]] + [[texture-units-xml|текстуры]] кнопок.
5. **Тексты** — [[loctable-xml|локализация]].

Не каждый мод трогает все слои: [[10x-скорость]] — только интерфейс,
[[второй-стражник]] — только `.def` + спавн.
