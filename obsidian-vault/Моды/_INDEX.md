# Индекс: Моды
Последнее обновление: 2026-06-05

Разбор каждого мода из коллекции `majesty_mods/`: что меняет, какие файлы
трогает, ключевые значения. У всех модов одинаковая установка (копировать
`resource.mod`) — см. [[установка-и-совместимость]].

## Моды
- [[10x-скорость]] — добавляет множители скорости `5x`/`10x` в HUD. Трогает только `game_window.xml`. Совместим со всеми.
- [[второй-стражник]] — второй лучник в башнях (`tower_guard`, `mks_tower_goblin`) со 2-го уровня через клон спавнера с условием `building_level>=2`. Трогает 2 `.def` + `global_spawn_settings.set`.
- [[элитные-башни]] — 3 новые уникальные башни (`unique_tower_guard/goblin/mage_statue`) в меню замка; ужесточает апгрейд обычной башни. Трогает `shops.set` + меню замка + текстуры.
- [[рыцарь]] — новый герой `hero_knight` в гильдии воинов, с умениями, оружием, бронёй и локализацией. Трогает `shops.set` + меню гильдии + loctable.
- [[4-кузница-у-монстров]] — 4-й уровень оружия/брони (`weapon05`/`bow05`) десяти героям фракции монстров (Unseen) + изобретения уровня 4. Трогает `inventions.graphml` + `shops.set` + кузницу.
- [[мини-сборка]] — большая сборка «зловещего» геймплея: ~31 логово монстров, гильдии, спавн, локализация миссий. Самый объёмный мод.

## Матрица переопределяемых файлов (для совместимости)
| Файл | Моды |
|---|---|
| `interface/.../game_window.xml` | 10x скорость |
| `gameData/spawn/global_spawn_settings.set` | второй стражник, мини-сборка |
| `set/trading/shops.set` | рыцарь, элитные башни, 4 кузница, мини-сборка |
| `interface/dynamic/menu/castle_menu.xml` | элитные башни, мини-сборка |
| `interface/dynamic/menu/guild_warrior_menu.xml` | рыцарь, мини-сборка |
| `entity/buildings/tower_guard/*.def` | второй стражник |

Моды, делящие файл, конфликтуют — подробнее в [[установка-и-совместимость]].
