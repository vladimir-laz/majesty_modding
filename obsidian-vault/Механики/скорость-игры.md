# Скорость игры

## Суть
Игрок управляет **скоростью течения игрового времени** через виджет в HUD.
Сам виджет (счётчик множителей) описан в
[[game-window-xml|game_window.xml]]; масштабирование времени по выбранному
множителю выполняет движок.

## Где живёт
- Контейнер `GAME_SPEED_CTRL` → панель `panel_game_speed` → счётчик
  `COUNTER_game_speed`.
- Список доступных скоростей — `<s_counter_values>` (через запятую).
- Кнопки −/+ переключают значение; `b_cycled=false` (нет зацикливания).

## Стандарт vs мод
Базовая игра ограничена низкими множителями. Мод [[10x-скорость]]
расширяет список до `0.2x,0.5x,1x,2x,5x,10x`, добавляя быстрые режимы
(до 10×). Это **чисто интерфейсная** правка — логику масштабирования
времени движок применяет сам по выбранному значению.
